Immaginiamo di dover creare una pagina per un blog con due tipi di paragrafi: 
uno "standard" e uno "evidenziato" (tipo una citazione o un riassunto).

1. La struttura HTML
Crea un file HTML e inserisci questo codice. Nota come usiamo l'attributo class per dare nomi specifici ai nostri elementi.

HTML
<div class="post-container">
  <h1 class="titolo-principale">Il mio viaggio in Giappone</h1>
  
  <p class="testo-giustificato">
    Questo è un paragrafo normale. Vogliamo che il testo sia distribuito bene su tutta la riga, 
    quindi useremo il giustificato per renderlo simile a una pagina di un libro.
  </p>

  <p class="testo-evidenziato">
    "In precedenza, non avrei mai immaginato di vedere i ciliegi in fiore a Tokyo."
  </p>

  <div class="box-bottoni">
    <button class="btn">Like</button>
    <button class="btn">Condividi</button>
    <button class="btn">Commenta</button>
  </div>
</div>

Adesso inserisci nel tag style il CSS per ogni classe presente nella struttura HTML

[√] la classe post-container deve avere le proprietà:

/* larghezza 80%; e margin auto; */

[√] la classe testo-giustificato deve avere le proprietà:

/* Allineamento del testo giustificato; e un colore #333; */

[√] la classe testo-evidenziato deve avere le proprità:

/* uno sfondo di colore coral; il bordo di sinistra con spessore 5px solid #007bff; padding 15px; e un margine top e bottom a 20px  */

[√] la classe box-bottoni deve avere le proprietà:

/* margine top 30px; */

[√] la classe btn deve avere:

/* padding top e bottom 10px e padding right e left 20px; sfondo blue; colore della scritta bianco; con un bordo arrotondato di 5px; il cursore; e il suo bordo deve essere eliminato */
